# comp352-a3
## Project
### Gradle / Gradlew
Execute the Gradle wrapper to have a local project version
- Windows: `gradlew`
- Linux/MacOS: `./gradlew`

#### Jar
`java -jar trees.jar`

### Build
- Windows: `gradlew build`
- Linux/MacOS: `./gradlew build`

### Jar
Builds are generated in `build/libs/`
- Windows: `gradlew jar`
- Linux/MacOS: `./gradlew jar`
